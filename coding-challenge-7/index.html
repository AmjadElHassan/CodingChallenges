Coding Challenge 7

Given an array of integers nums and an integer target, return indices of the two numbers such that they add up to target. You may assume that each input would have exactly one solution, and you may not use the same element twice. You can return the answer in any order.

use console function twoSum([array of numbers],target-number)

Developer notes: seemingly pretty simple, i actually spent almost an hour and half on this problem, frustratingly enough. The issue i was having was based on the limitations of certain array methods that i tried to make work.

Most specifically, arr.indexOf() has an issue where it will only EVER return the first instance of an array element that matches that value. so in the instance of the array ([7,5,7],14), it would return [0,0] rather than [0,2] because they are both of the value 7.

So to circumvent this i created an index of the indeces of every item in the array of numbers, then i created a for loop that would compare the item of every single item by cycling through the index. this is the method i used to keep account of every item in the array. next, i created a loop within that that goes through each item in the array as well.

if the variables i created, i and j, representing the index of the number i'm testing vs. every other number in the array, are the same, it means we are on the same element, and it will return a null result. otherwise, it will test if their sum equals the target, if so it will return the value of the index of the number being tested
